Η οικογένεια Φυτρολάκη σας καλωσορίζει στα ενοικιαζόμενα διαμερίσματα "Fytrolakis Apartments".

Τα Διαμερίσματα μας βρίσκονται στον Ξερόκαμπο Ζήρου Σητείας στο νοτιοανατολικότερο άκρο της Κρήτης, απέχουν μόλις 50μ από την ακτή και έχουν υπέροχη θέα προς την θάλασσα.

Σας περιμένουμε για να σας φιλοξενήσουμε και σας υποσχόμαστε οτι θα απολαύσετε τις διακοπές σας σε ένα όμορφο, καθαρό και άνετο περιβάλλον.
